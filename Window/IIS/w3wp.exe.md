# w3wp.exe

Microsoft Corporation 의 인터넷 정보 서비스 (IIS) 소프트웨어 구성 요소입니다.

### 점유 도메인 확인방법

```
C:\Windows\System32\inetsrv
```

appcmd.exe 발견 후

```
appcmd list wp
```
